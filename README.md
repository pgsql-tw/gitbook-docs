# PostgreSQL 正體中文使用手冊

本專案用於編譯 PostgreSQL 官方使用手冊之正體中文版  
主要服務台灣的 PostgreSQL 使用者  
開始閱讀：[https://docs.postgresql.tw/](https://docs.postgresql.tw/)

目前編譯的語言及版本為:

* 台灣正體中文, PostgreSQL 10
