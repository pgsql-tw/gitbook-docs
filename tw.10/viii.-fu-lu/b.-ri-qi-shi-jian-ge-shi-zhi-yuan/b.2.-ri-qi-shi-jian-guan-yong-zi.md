**Table B.3. 日期/時間修飾字**

| 修飾字 | 說明 |
| :--- | :--- |
| `AM` | 12:00 之前的時間 |
| `AT` | 忽略 |
| `JULIAN`,`JD`,`J` | 接下來的字串是 Julian Date 格式 |
| `ON` | 忽略 |
| `PM` | 12:00 之後的時間 |
| `T` | 接下來的字串是 time |
